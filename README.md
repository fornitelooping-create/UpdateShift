# EchoTalk

## Prerequisites

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies: `npm install`.

## Run Locally (web / Vite dev server)

```bash
npm run dev
```

Open the local URL printed by Vite.

## Build (web)

```bash
npm run build
npm run preview
```

## Run With Electron

See [ELECTRON_SETUP.md](./ELECTRON_SETUP.md) for the full Electron development and packaging workflow (`vite.electron.config.js`, `electron/main.cjs`, `electron/preload.cjs`).
